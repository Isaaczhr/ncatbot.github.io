帮助文档

  1. 下载NapCat 找到 NapCat.Shell.zip 并解压
  2. 启动NapCat：
    • 第一次启动机器人：直接双击 launcher.bat 文件
    • 启动已经扫码登录过的机器人，在当前目录地址栏输入 cmd 或右键打开命令行，在当前目录打开 cmd 命令行，输入 start launcher.bat [你的QQ号]
  3. 部分使用参考代码可查看 main.py
  4. example 文件夹里是每个经过测试的 api 类别例子，没有标注其他情况的是完全执行成功过的，可以使用异步正常插入代码中
  5. 每个 api 提供了有限帮助支持（在编辑器里按住 Ctrl 并单击 api 方法即可跳转）
  6. 部分教程可查看 NapCatQQ开发机器人PythonSDK
  7. 如果可以，希望你可以帮助完善😄
